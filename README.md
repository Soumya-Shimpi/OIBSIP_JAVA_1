Online Reservation System (Java)
Overview
The Online Reservation System is a Java-based application developed as part of an internship task. The project simulates a basic railway reservation system that allows users to book and cancel tickets efficiently while maintaining data in a centralized structure.
This project demonstrates core programming concepts, system design, and modular development using Java.
Objective
• To design a system that simplifies ticket reservation and cancellation
• To apply Object-Oriented Programming (OOP) concepts in a real-world scenario
Features
• User Login Authentication
• Ticket Reservation System
• Automatic PNR Generation
• Ticket Cancellation using PNR
• Console-based User Interface
Modules
1. Login Module
• Validates user credentials
• Grants access to the system upon successful authentication
2. Reservation Module
• Accepts user details
• Auto-fills train name based on train number
• Generates a unique PNR
• Stores booking details
3. Cancellation Module
• Accepts PNR number
• Displays booking details
• Confirms and processes cancellation
